# OpenClaw GitHub 相关技能列表

## 技能商店搜索结果

根据 skillhub search github 的结果，以下是 OpenClaw 社区中可用的 GitHub 相关技能：

### 1. github-api (已安装)
- **版本**: 1.0.3
- **描述**: GitHub API integration with managed OAuth. Access repositories, issues, pull requests, commits, branches, and users.
- **功能**: GitHub REST API 集成，支持托管 OAuth，访问仓库、issues、PR、提交、分支和用户
- **适合场景**: 需要与 GitHub 仓库交互、管理 issues 和 PR、搜索代码或自动化工作流
- **当前状态**: 已安装在 `/root/.openclaw/workspace/skills/github-api/SKILL.md`

### 2. github
- **版本**: 1.0.0
- **描述**: Interact with GitHub using the `gh` CLI. Use `gh issue`, `gh pr`, `gh run`, and `gh api` for issues, PRs, CI runs, and advanced queries.
- **功能**: 使用 `gh` CLI 与 GitHub 交互，通过 `gh issue`、`gh pr`、`gh run` 和 `gh api` 管理 issues、PR、CI 运行及高级查询
- **安装命令**: `skillhub install github`

### 3. github-ai-trends
- **版本**: 1.1.0
- **描述**: Generate GitHub AI trending project reports as formatted text leaderboards. Fetches top-starred AI/ML/LLM repos by daily, weekly, or monthly period and renders a styled leaderboard.
- **功能**: 生成 GitHub AI 趋势项目的文本排行榜，按日、周、月抓取 AI/ML/LLM 高星仓库并渲染
- **适合场景**: 查询 AI 项目趋势、GitHub 趋势、AI 排行榜或热门 AI 仓库
- **安装命令**: `skillhub install github-ai-trends`

### 4. github-cli
- **版本**: 1.0.0
- **描述**: Comprehensive GitHub CLI (gh) reference. Covers repos, issues, PRs, Actions, releases, gists, search, projects v2, API, secrets/variables, labels, codespaces...
- **功能**: GitHub CLI (gh) 综合参考指南，涵盖仓库、Issue、PR、Actions、发布、Gist、搜索、Projects v2、API、密钥/变量、标签、Codespaces 等
- **安装命令**: `skillhub install github-cli`

### 5. github-trending-cn
- **版本**: 1.0.0
- **描述**: GitHub 趋势监控 | GitHub Trending Monitor. 获取 GitHub 热门项目、编程语言趋势、开源动态
- **功能**: GitHub 趋势监控，获取 GitHub 热门项目、编程语言趋势、开源动态
- **适合场景**: GitHub、trending、开源、热门项目查询
- **安装命令**: `skillhub install github-trending-cn`

### 6. github-kb
- **版本**: 1.0.1
- **描述**: Manage a local GitHub knowledge base and provide GitHub search capabilities via gh CLI.
- **功能**: 管理本地 GitHub 知识库，提供 gh CLI 搜索功能
- **适合场景**: 用户询问仓库、PR、Issue、克隆、代码库或项目信息时使用
- **安装命令**: `skillhub install github-kb`

### 7. github-mcp
- **版本**: 1.0.0
- **描述**: GitHub MCP Server enables AI agents to manage repos, read/update files, handle issues/PRs, branches, and automate GitHub workflows via the API.
- **功能**: GitHub MCP Server 让 AI 代理能够管理仓库、读写文件、处理 Issue/PR、分支，并通过 API 自动化 GitHub 工作流
- **安装命令**: `skillhub install github-mcp`

### 8. github-issue-creator
- **版本**: 0.1.0
- **描述**: Convert raw notes, error logs, voice dictation, or screenshots into crisp GitHub-flavored markdown issue reports.
- **功能**: 将原始笔记、错误日志、语音或截图转为简洁的 GitHub 风格 Markdown issue 报告
- **适合场景**: 用户粘贴错误信息或随意描述并希望生成结构化 GitHub issue 的场景，支持图片/GIF 证据
- **安装命令**: `skillhub install github-issue-creator`

### 9. github-token
- **版本**: 1.0.0
- **描述**: Interact with GitHub using Personal Access Tokens. Secure, user-controlled access - no OAuth, no full account access.
- **功能**: 使用个人访问令牌与 GitHub 交互，提供安全、用户可控的访问权限，无需 OAuth，也无完整账户访问权
- **适合场景**: 用户操作 GitHub 仓库的场景，支持克隆、推送、分支、PR 及 Issues

## 技能分类

### 基础技能（必需）
- **github-api**: REST API 集成（已安装）
- **github**: gh CLI 工具
- **github-token/github-pat**: 个人访问令牌操作

### 自动化技能
- **github-ops**: 全自动 GitHub 操作
- **github-automation-pro**: 高级自动化
- **github-mcp**: MCP 服务器集成

### 开发辅助技能
- **github-issue-creator**: 创建结构化 Issue
- **github-actions-troubleshooting**: Actions 故障排查
- **github-actions-generator**: Actions 工作流生成
- **github-copilot-cli**: Copilot CLI 高效使用

### 信息查询技能
- **github-ai-trends**: AI 趋势分析
- **github-trending**: 热门仓库查询
- **github-trending-cn**: 中国用户趋势监控
- **github-search**: 深度搜索分析
- **github-kb**: 知识库管理

### 问题解决技能
- **github-issue-resolver**: 自主 Issue 解决

### 工具集成技能
- **github-image-hosting**: 图片托管
- **github-mpc**: MCP 服务器配置

## 推荐安装顺序

1. **基础**: `github` (gh CLI) + `github-api` (REST API) - 已安装
2. **自动化**: `github-automation-pro` - 高级自动化功能
3. **开发**: `github-actions-generator` - CI/CD 工作流生成
4. **信息**: `github-trending` - 热门项目查询
5. **问题**: `github-issue-creator` - Issue 创建助手

## 使用建议

1. **初学者**: 先安装 `github` 和 `github-api` 进行基础操作
2. **开发者**: 安装 `github-actions-generator` 和 `github-copilot-cli`
3. **项目经理**: 安装 `github-automation-pro` 和 `github-issue-creator`
4. **研究人员**: 安装 `github-ai-trends` 和 `github-trending`
5. **AI 助手**: 全栈安装，实现完整的 GitHub 管理自动化

## 安装方法

使用 skillhub（中国优化版本）进行安装：

```bash
skillhub install github
skillhub install github-automation-pro
skillhub install github-actions-generator
```

如果 skillhub 不可用或限制，可以尝试 clawhub（公共注册中心）：

```bash
clawhub install github
clawhub install github-automation-pro
```

**注意**: 根据政策，优先尝试 `skillhub`，如果不可用再尝试 `clawhub`。
