🚀 为什么选择 Node Magic？

Node Magic 专为需要高效管理多个订阅的用户打造。 无论你使用 Clash、Surge、Shadowrocket、Stash、Quantumult X，都能一键合并、去重、优化。

❌ 痛点

  • 多个订阅来源，手动合并很麻烦
  • 节点重复太多，影响使用体验
  • 每次换设备都要重新配置
  • 想自建，但不会写代码
  • 订阅转换工具不稳定、速度慢

✅ 解决方案

  • 自动合并所有订阅
  • 智能指纹级去重
  • Cloudflare KV 持久化配置
  • 可视化 Web 面板，无需写代码
  • 全球 CDN 加速，稳定高速
  • 完全开源，永久免费

🛠️ 3 分钟完成部署

1

Fork 项目

点击 GitHub 的 Fork 按钮，获取你的专属版本仓库。

2

一键部署到 Cloudflare Worker

无需服务器、无需域名、无需环境配置。直接在 Cloudflare Dashboard 创建 Worker 并粘贴代码即可。

3

配置并开始使用

打开 Web 面板 → 添加订阅链接 → 设置优选规则 → 获取最终订阅链接，导入客户端即可。

🎨 可视化配置界面

通过直观的 Web 界面，轻松管理订阅、设置去重规则，所有操作实时生效。

📋 订阅管理

添加、编辑、删除订阅源，一键测试可用性

⚙️ 去重设置

灵活配置去重规则，保留优先级最高的节点

💾 KV 状态

实时查看 Cloudflare KV 存储状态，配置自动同步

🔗 订阅输出

一键生成最终订阅链接，即时预览合并结果

🔐 安全与隐私

Node Magic 不会存储你的任何订阅内容。
所有处理均在你的 Cloudflare Worker 中本地完成，数据和配置完全由你掌控，安全可靠。
